# Jupyter Notebooks 

This folder contains Jupyter notebooks for the nanopore data analysis and the version control with Git module. 

## Nanopore Analysis
- [**DSC_Workshop2026_Mod_2_Epi2meSetup.ipynb**](Nanopore_Analysis/DSC_Workshop2026_Mod_2_Epi2meSetup.ipynb) - Install & test Nextflow to use with Epi2me pipelines

- [**DSC_Workshop2026_Mod_2_wf-metagenomics.ipynb**](Nanopore_Analysis/DSC_Workshop2026_Mod_2_wf-metagenomics.ipynb) - Runs the Epi2me metagenomics workflow including Taxonomic Classification and antimicrobial Resistance (AMR) Detection.

- [**DSC_Workshop2026_Mod_2_wf-bacterial-genomes.ipynb**](Nanopore_Analysis/DSC_Workshop2026_Mod_2_wf-bacterial-genomes.ipynb) - Runs the Epi2me bacterial-genomes workflow including genome assembly & annotation, and Isolate Characterization.
- [**DSC_Workshop2026_Mod_4_install_bioinfo_tools.ipynb**](Nanopore_Analysis/DSC_Workshop2026_Mod_4_install_bioinfo_tools.ipynb) - Walkthrough of the install_bioinfo_tools.sh script.

- [**DSC_Workshop2026_Mod_4_process_AS.ipynb**](Nanopore_Analysis/DSC_Workshop2026_Mod_4_process_AS.ipynb) - This document provides a step-by-step walkthrough of the adaptive sampling analysis pipeline.

- [**DSC_Workshop2026_Mod_4_get_stats.ipynb**](Nanopore_Analysis/DSC_Workshop2026_Mod_4_get_stats.ipynb) - Walkthrough of the get_all_stats.sh script.

## Git  
- [**DSC_Workshop2026_Mod_3_Lesson1_Git.ipynb**](Git/DSC_Workshop2026_Mod_3_Lesson1_Git.ipynb) - This notebook includes configuring your git environment.

- [**DSC_Workshop2026_Mod_3_Lesson2_Git_and_Github.ipynb**](Git/DSC_Workshop2026_Mod_3_Lesson2_Git_and_Github.ipynb) - This notebook includes creating your first version-controlled project, learning how to stage and commit a file, make changes to this file, view the history of your project, and create the .gitignore file. 

- [**DSC_Workshop2026_Mod_3_Lesson3_Collaborating_Collaborator1.ipynb**](Git/DSC_Module_3_Lesson3_Collaborating_Collaborator1.ipynb) - This notebook is part of the collaboration on Github Lesson and contains instructions for Collaborator 1.

- [**DSC_Workshop2026_Mod_3_Lesson3_Collaborating_Collaborator2.ipynb**](Git/DSC_Workshop2026_Mod_3_Lesson3_Collaborating_Collaborator1.ipynb) - This notebook is part of the collaboration on Github Lesson and contains instructions for Collaborator 2.

- [**DSC_Workshop2026_Mod_3_Lesson4_RStudioServer_Git.ipynb**](Git/DSC_Workshop2026_Mod_3_Lesson4_RStudioServer_Git.ipynb) - This notebook includes how to clone your repository into RStudio Server and use RStudio's built-in Git tools to commit and push changes back to your GitHub repository.
